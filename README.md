# Description
This project is the backend service for implementing these domain features in Temuka Platform and Temuka Partner Dashboard:
1. Feed Recommendation 
2. Search System
3. Social Network Recommendation
4. Major, Community, and University Stats

# Tech Stack
- Java (Programming language)
- Spring Boot (Web framework)
- MongoDB (Non relational database)
- Redis (Key value store)
- RabbitMQ (Message queue)
- ElasticSearch (Distributed search and analytics engine)

# Command
1. Run the server
2. Run the consumer
3. Run the migration
4. Run the unit test
